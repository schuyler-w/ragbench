# ragbench

A retrieval-augmented generation (RAG) evaluation framework built using 50,000 Wikipedia articles. The focus is to a RAG pipeline and to systematically measure and emperically improve it. We will evaluate retrieval quality across embedding models, chunk sizes, and overlap configurations using a synthetic QA dataset generated by `llama3.1:70b`.
